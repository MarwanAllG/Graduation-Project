Welcome to Used Cars Analysis and Prediction Application!!


Firstly, this is a graduation project done by 4 students majoring in data science at the University of Jeddah.

The web application is aimed to help individuals and enterprises that are willing to buy or sell used cars in Saudi Arabia to set prices for cars in the most accurate way.

This is done by taking input from the website’s users, then processing it using a machine learning model that predicts the car price based on the user’s input.

Furthermore, the R squared score of the model used to predict the car prices is ≈ %96 , hence you can rest assured that the prediction is quite much accurate 😊 .

Additionally, the web application will route its users to an interactive dashboard that will help in understanding the used cars market in Saudi Arabia.
